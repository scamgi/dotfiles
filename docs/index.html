One-Step Installation

To get started, copy and paste the following command into your terminal and press Enter. The script will automatically detect your operating system (macOS or Linux) and run the appropriate installer.

curl -fsSL https://raw.githubusercontent.com/scamgi/dotfiles/main/install | bash

Check out the repository on GitHub for more details.
